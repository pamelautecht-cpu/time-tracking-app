# Company Setup Checklist

Use this checklist to get your employee time tracking system up and running quickly.

---

## ‚úÖ Initial Setup (One-Time)

### Step 1: Access Your Application
- [ ] Open app URL from Creao dashboard
- [ ] Verify app loads successfully
- [ ] Bookmark URL for easy access

### Step 2: Configure Settings
- [ ] Select preferred language (EN/ES)
- [ ] Test language switching
- [ ] Confirm language preference saves

### Step 3: Create Manager Account
- [ ] Go to "Employees" tab
- [ ] Click "Add Employee"
- [ ] Enter your information:
  - [ ] Name
  - [ ] Email
  - [ ] Phone (optional)
  - [ ] Role: **Manager**
- [ ] Save and verify manager appears in list

### Step 4: Add All Employees
For each employee, repeat:
- [ ] Click "Add Employee"
- [ ] Enter employee information:
  - [ ] Full name
  - [ ] Email address
  - [ ] Phone number (optional)
  - [ ] Role: **Employee** (or Manager if applicable)
- [ ] Save
- [ ] Verify employee appears in list

**Tip:** Have employee information ready (name, email, phone) before starting

### Step 5: Test the System
- [ ] Select an employee account
- [ ] Click "Clock In"
- [ ] Verify timer starts
- [ ] Add test location
- [ ] Add test work description
- [ ] Click "Clock Out"
- [ ] Verify time card appears in history
- [ ] Switch to manager account
- [ ] Verify test time card appears in "Time Cards" tab
- [ ] Delete or keep test entry

## üìÖ Weekly Operations Checklist

### Monday Morning
- [ ] Switch to "Reports" tab
- [ ] Select previous week (Monday date)
- [ ] Review weekly summary
- [ ] Check for overtime (40+ hours)
- [ ] Review detailed time cards
- [ ] Verify all locations and descriptions are present

### Generate Payroll Report
Choose one or both:

**Option 1: Print Report**
- [ ] Click "Print Report" button
- [ ] Review print preview
- [ ] Adjust printer settings if needed
- [ ] Print report
- [ ] Submit to payroll department

**Option 2: Export CSV**
- [ ] Click "Export to CSV" button
- [ ] Locate downloaded file
- [ ] Open in Excel/Sheets to verify
- [ ] Send file to payroll system
- [ ] Archive copy for records

### Review & Follow-up
- [ ] Note any overtime concerns
- [ ] Address any missing data
- [ ] Follow up with employees about incomplete entries
- [ ] Archive weekly report

## üéØ Troubleshooting Quick Fixes

| Issue | Quick Fix |
|-------|-----------|
| Employee forgot to clock in | Manager adds manual time card |
| Employee forgot to clock out | Manager edits time card to add clock out |
| Wrong time recorded | Manager edits time card |
| Missing location/description | Manager edits time card to add info |
| Overtime unexplained | Review detailed report, contact employee |
| Can't print | Try different browser, check printer |
| CSV won't download | Check browser settings, allow downloads |
| Language won't change | Refresh page, clear cache |

## üö® Common Setup Mistakes to Avoid

‚ùå **DON'T:**
- Skip employee training
- Forget to test the system first
- Add employees without verifying email addresses
- Wait until payroll deadline to generate first report
- Delete employees with existing time cards

‚úÖ **DO:**
- Train employees thoroughly on first day
- Test with sample time cards before going live
- Verify all employee information is correct
- Generate test reports before first real payroll
- Archive data before making major changes
