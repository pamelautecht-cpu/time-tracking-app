### For First-Time Setup (Manager)

#### Step 1: Access the Application
```
1. Open the app URL in your web browser
2. You'll see the employee time tracking system homepage
```

#### Step 2: Choose Your Language
```
1. Look in the top-right corner
2. Click the language selector (EN/ES)

```

#### Step 3: Select or Create Manager Account
```
1. Click "Select User"
2. If you're the first user, you may need to add yourself:
   - Go to "Employees" tab
   - Click "Add Employee"
   - Enter your name, email, phone
   - Set role to "Manager"
   - Click save
```

#### Step 4: Add Your Employees
```
1. Go to "Employees" tab
2. Click "Add Employee" button
3. Fill in:
   - Name (required)
   - Email (required)
   - Phone (optional)
   - Role: Select "Employee" or "Manager"
4. Click save
5. Repeat for all employees
```

## üìä Understanding Your Data

### How Hours Are Calculated

**Regular Hours:**
```
- First 40 hours worked in a week (Monday-Sunday)
- Lunch breaks are automatically deducted
- Shown in "Regular Hours" column in reports
```

**Overtime Hours:**
```
- Any hours beyond 40 in a week
- Highlighted in orange in reports
- Shown in "Overtime Hours" column
- Badge shows "Overtime Alert"
```

**Total Hours:**
```
Total Hours = Regular Hours + Overtime Hours
```

**Example:**
```
If employee works 45 hours in a week:
- Regular Hours: 40.00 hrs
- Overtime Hours: 5.00 hrs (highlighted in orange)
- Total Hours: 45.00 hrs
```
