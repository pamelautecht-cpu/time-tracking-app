## üÜò Common Sharing Questions

**Q: Do I need to install anything?**
A: No! It's a web app. Just share the URL.

**Q: Will it work on their phones?**
A: Yes! Works on all devices with a web browser.

**Q: Do they need to create accounts?**
A: No! Managers add employees to the system, then employees just select their name.

**Q: What if they forget the URL?**
A: They can bookmark it, add to home screen, or you can create a QR code.

**Q: Can multiple companies use the same app?**
A: Each company gets their own instance/URL from Creao. Data is isolated.

**Q: How do I give someone access?**
A: Managers add employees in the "Employees" tab. Then share the app URL.

**Q: What if someone loses their phone?**
A: They can access from any device using the same URL.

**Q: Is their data safe?**
A: Yes! All data is stored in Creao's persistent database with regular backups.
